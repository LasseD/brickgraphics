The BrickGraphics project is an open source project where anyone can contribute.

BrickGraphics currently consists of the following applications:
- Bricktilism
- Griddy

Please remember to add your name to the @author attribute when contributing to a .java file. 
All contributers must receive credit in about-dialogs, names ordered lexicographically by family name,first name. Authors are added in authors.txt

Please preserve the code style (indentation and placement of curly braces and so on) when contributing to a .java file, but feel free to use your preferred style when adding new files.

Please contact Lasse Deleuran (lassedeleuran@gmail.com) if you would like to contribute to BrickGraphics.
